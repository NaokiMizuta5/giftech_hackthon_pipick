# giftech-hakathon-team-d

## Installation

### Install Bun

```bash
curl hhtps://bun.sh/install | bash
```

```bash
source ~/.zshrc
```

### Clone Repository

```bash
git clone git@github.com:aose-yuu/giftech-hakathon-team-d.git
```

### Install Packages

```bash
bun install
```

### Load Lefthook

```bash
bun run lefthook:install
```

### Start Project

```bash
bun run start
```

Scan a QR Code with your iPhone or Android device.
